# práctica-web-wordpress
# Práctica de página web con wordpress

## Sobre mí 😊👇
- Estudiante de Desarrollo de Aplicaciones Multiplataforma
- Junior Developer en formación
- Manejo de Java, Php y HTML
- Experiencia con Apache NetBeans, PhpStorm, Eclipse, phpMyAdmin, mySQL, Oracle VirtualBox y Docker.

## Objetivos 🎯

- Instalar y configurar un sitio web utilizando WordPress en un entorno local, una 
máquina virtual o un hosting de pruebas. 
- Crear una página web básica para una empresa ficticia llamada: TecnoAula Formación. La empresa se dedica a ofrecer formación tecnológica para estudiantes, trabajadores y empresas. 

## Requisitos de la práctica 📑

1. Instalación de WordPress

● Preparar un entorno con Apache y MySQL/MariaDB.<br> 
● Descargar WordPress desde la web oficial. <br>
● Crear una carpeta para el proyecto.<br> 
● Crear una base de datos para WordPress.<br> 
● Completar el asistente de instalación.<br> 
● Acceder correctamente al panel de administración. <br>

2. Datos mínimos del sitio 

● Nombre del sitio: TecnoAula Formación <br>
● Descripción corta relacionada con formación tecnológica. <br>
● Usuario administrador creado correctamente. <br>
● Contraseña segura. <br>
● Idioma configurado en español. <br>

3. Configuración básica 

● Título del sitio. <br>
● Descripción corta. <br>
● Zona horaria. <br>
● Formato de fecha y hora. <br>
● Enlaces permanentes en formato Nombre de la entrada. <br>
● Página de inicio establecida correctamente. <br>

4. Páginas obligatorias 

● Inicio <br>
● Sobre nosotros <br>
● Servicios <br>
● Contacto <br>

5. Contenido mínimo de las páginas 

La página de Inicio deberá incluir: <br>
● Nombre de la empresa. <br>
● Texto de bienvenida. <br>
● Imagen o cabecera. <br>
● Breve descripción de los servicios.

La página Sobre nosotros deberá incluir: <br>
● Historia ficticia de la empresa. <br>
● Misión. <br>
● Valores. <br>

La página Servicios deberá incluir al menos tres servicios: <br>
● Formación en informática básica. <br>
● Cursos de redes y sistemas. <br>
● Formación para empresas. <br>

La página Contacto deberá incluir: <br>
● Dirección ficticia. <br>
● Teléfono ficticio. <br>
● Correo ficticio. <br>
● Formulario de contacto o simulación visual. <br>

6. Menú de navegación
   
El sitio deberá tener un menú principal visible con enlaces a: <br>
● Inicio <br>
● Sobre nosotros <br>
● Servicios <br>
● Contacto <br>

7. Tema 

● Instalar un tema gratuito. <br>
● Activarlo correctamente. <br>
● Personalizar mínimamente el aspecto visual del sitio. <br>
Temas recomendados: <br>
● Astra <br>
● OceanWP <br>
● GeneratePress <br>
● Kadence <br>
● Twenty Twenty-Five <br>

8. Plugins
   
Se deberán instalar y activar al menos dos plugins. <br>
Plugins recomendados: <br>
● Contact Form 7 <br>
● Yoast SEO <br>
● WP Mail SMTP <br>
● UpdraftPlus <br>
● Wordfence Security <br>
● LiteSpeed Cache <br>
Indicar brevemente para qué sirve cada plugin instalado. <br>

9. Seguridad básica

● No utilizar una contraseña débil. <br>
● Mantener WordPress, tema y plugins actualizados. <br>
● Eliminar plugins innecesarios. <br>
● Eliminar temas no utilizados, dejando al menos uno de respaldo. <br>
● Crear un usuario adicional con rol de Editor o Autor. <br>

10. Copia de seguridad
    
Realizar una copia de seguridad del sitio mediante uno de estos métodos: <br>
● Exportación de la base de datos desde phpMyAdmin. <br>
● Plugin de copias de seguridad. <br>
● Copia manual de archivos y base de datos. <br>

11. Entrega obligatoria

● Memoria técnica en PDF. <br>
● Capturas del proceso. <br>
● Exportación de la base de datos. <br>
● Archivo comprimido del proyecto si se ha trabajado en local. <br>

12. Capturas mínimas requeridas
    
La memoria deberá incluir capturas de: <br>
● Carpeta del proyecto. <br>
● Base de datos creada. <br>
● Instalación de WordPress. <br>
● Panel de administración. <br>
● Ajustes generales. <br>
● Enlaces permanentes.<br>
● Tema instalado. <br>
● Plugins instalados. <br>
● Páginas creadas. <br>
● Menú creado. <br>
● Vista final de la página web. <br>
● Copia de seguridad realizada.<br>
