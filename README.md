# práctica-web-wordpress
Práctica de página web con wordpress
Debes instalar y configurar un sitio web utilizando WordPress en un entorno local, una 
máquina virtual o un hosting de pruebas. 
Una vez instalado WordPress, deberás crear una página web básica para una empresa 
ficticia llamada: 
TecnoAula Formación 
La empresa se dedica a ofrecer formación tecnológica para estudiantes, trabajadores y 
empresas. 
La web deberá estar correctamente configurada, tener una estructura mínima de páginas, 
un menú de navegación funcional, un tema instalado y al menos dos plugins configurados. 
También deberás aplicar algunas medidas básicas de seguridad y realizar una copia de 
seguridad del sitio.

Requisitos de la práctica 
1. Instalación de WordPress 
El alumno deberá: 
● Preparar un entorno con Apache y MySQL/MariaDB. 
● Descargar WordPress desde la web oficial. 
● Crear una carpeta para el proyecto. 
● Crear una base de datos para WordPress. 
● Completar el asistente de instalación. 
● Acceder correctamente al panel de administración. 
2. Datos mínimos del sitio 
El sitio deberá tener los siguientes datos: 
● Nombre del sitio: TecnoAula Formación 
● Descripción corta relacionada con formación tecnológica. 
● Usuario administrador creado correctamente. 
● Contraseña segura. 
● Idioma configurado en español. 
3. Configuración básica 
Se deberá configurar: 
● Título del sitio. 
● Descripción corta. 
● Zona horaria. 
● Formato de fecha y hora. 
● Enlaces permanentes en formato Nombre de la entrada. 
● Página de inicio establecida correctamente. 
4. Páginas obligatorias 
La web deberá contener, como mínimo, las siguientes páginas: 
● Inicio 
● Sobre nosotros 
● Servicios 
● Contacto 
Cada página deberá tener contenido coherente con la empresa ficticia. 
5. Contenido mínimo de las páginas 
La página de Inicio deberá incluir: 
● Nombre de la empresa. 
● Texto de bienvenida. 
● Imagen o cabecera. 
● Breve descripción de los servicios. 
La página Sobre nosotros deberá incluir: 
● Historia ficticia de la empresa. 
● Misión. 
● Valores. 
La página Servicios deberá incluir al menos tres servicios: 
● Formación en informática básica. 
● Cursos de redes y sistemas. 
● Formación para empresas. 
La página Contacto deberá incluir: 
● Dirección ficticia. 
● Teléfono ficticio. 
● Correo ficticio. 
● Formulario de contacto o simulación visual. 
6. Menú de navegación 
El sitio deberá tener un menú principal visible con enlaces a: 
● Inicio 
● Sobre nosotros 
● Servicios 
● Contacto 
7. Tema 
El alumno deberá: 
● Instalar un tema gratuito. 
● Activarlo correctamente. 
● Personalizar mínimamente el aspecto visual del sitio. 
Temas recomendados: 
● Astra 
● OceanWP 
● GeneratePress 
● Kadence 
● Twenty Twenty-Five 
8. Plugins 
Se deberán instalar y activar al menos dos plugins. 
Plugins recomendados: 
● Contact Form 7 
● Yoast SEO 
● WP Mail SMTP 
● UpdraftPlus 
● Wordfence Security 
● LiteSpeed Cache 
El alumno deberá indicar brevemente para qué sirve cada plugin instalado. 
9. Seguridad básica 
Se deberán aplicar las siguientes medidas: 
● No utilizar una contraseña débil. 
● Mantener WordPress, tema y plugins actualizados. 
● Eliminar plugins innecesarios. 
● Eliminar temas no utilizados, dejando al menos uno de respaldo. 
● Crear un usuario adicional con rol de Editor o Autor. 
10. Copia de seguridad 
El alumno deberá realizar una copia de seguridad del sitio mediante uno de estos métodos: 
● Exportación de la base de datos desde phpMyAdmin. 
● Plugin de copias de seguridad. 
● Copia manual de archivos y base de datos. 
11. Entrega obligatoria 
El alumno deberá entregar: 
● Memoria técnica en PDF. 
● Capturas del proceso. 
● Exportación de la base de datos. 
● Archivo comprimido del proyecto si se ha trabajado en local. 
12. Capturas mínimas requeridas 
La memoria deberá incluir capturas de: 
● Carpeta del proyecto. 
● Base de datos creada. 
● Instalación de WordPress. 
● Panel de administración. 
● Ajustes generales. 
● Enlaces permanentes. 
● Tema instalado. 
● Plugins instalados. 
● Páginas creadas. 
● Menú creado. 
● Vista final de la página web. 
● Copia de seguridad realizada.
